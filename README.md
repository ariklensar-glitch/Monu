Python Mini Projects Collection

Is repository mein mere Python seekhne ke safar ke 5 shuruati projects hain. Ye projects logic building aur basic programming concepts ko samajhne ke liye banaye gaye hain.

Projects ki List

1.Number Guessing Game (`Number_gussing.py`)
Is game mein computer ek random number chunta hai aur aapko use guess karna hota hai.
**Concepts:** `random` module, `while` loop, `if-else` conditions.

2.Basic Calculator (`Basic_calculator.py`)
Ek simple calculator jo basic math operations (Add, Subtract, Multiply, Divide) karta hai.
**Concepts:** Functions, User Input, Arithmetic Operators.

3.To-Do List (`TO_DO_LIST.py`)
Apne rozana ke kaamo ko manage karne ke liye ek command-line based list.
**Concepts:** Lists, List Methods (`append`, `remove`), Loops.

4. Password Generator
Ek strong aur random password generate karne wala program.

Computer ke khilaaf ek classic game. (Coming Soon!)
